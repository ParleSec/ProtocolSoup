# GHCR Services Overview

ProtocolSoup publishes service images to GitHub Container Registry.

## P0 Services

- `protocolsoup-gateway`
- `protocolsoup-federation`
- `protocolsoup-vc`
- `protocolsoup-scim`
- `protocolsoup-ssf`
- `protocolsoup-spiffe`
- `protocolsoup-frontend`
- `protocolsoup-wallet`
- `protocolsoup-spire-server`
- `protocolsoup-spire-agent`
- `protocolsoup-spire-registration`

## POC Evaluation Focus

- Can the platform support one canonical page per image with:
  - runtime purpose,
  - environment variables,
  - health/readiness checks,
  - security hardening guidance,
  - troubleshooting patterns?
