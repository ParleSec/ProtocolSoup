# API Reference Overview

This section evaluates API-first documentation quality for two representative services:

- `gateway`: aggregated protocol index, health and routing helpers.
- `scim`: RFC 7644 provisioning endpoints and discovery resources.

## What We Are Testing

- Endpoint discoverability and navigation depth.
- Request and response readability.
- Authentication and content-type clarity.
- Search relevance for endpoint-oriented queries.

The OpenAPI sources used by this POC are in `openapi/gateway.yaml` and `openapi/scim.yaml`.
