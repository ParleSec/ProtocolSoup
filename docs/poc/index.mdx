# ProtocolSoup Documentation POC

This is the initial public docs IA skeleton used to evaluate docs platform candidates.

## What This POC Covers

- Public onboarding and product overview paths.
- GHCR service documentation structure.
- API reference generation from OpenAPI.
- Protocol and standards documentation structure.

## Representative APIs In Scope

- `gateway` (aggregation and routing behavior)
- `scim` (resource model and provisioning operations)
