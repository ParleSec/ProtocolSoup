# Start Here

ProtocolSoup is an interactive protocol learning and integration platform that runs real protocol flows and exposes inspectable traffic and artifacts.

## Audience Paths

- **Users:** Learn protocol behavior and run guided flows in Looking Glass.
- **Developers:** Integrate against API surfaces and deploy GHCR services.

## Quick Navigation

- See `GHCR Services` for deployment and runtime contracts.
- See `API Reference` for endpoint-level contracts.
- See `Protocols And Specs` for standards context and flow semantics.
